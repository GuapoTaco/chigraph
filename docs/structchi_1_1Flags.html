Detailed Description

template<typename Enum>
struct chi::Flags< Enum >

A template class for type-safe flags.

Usage:

enum class MyOpts {
First = 1u,
Second = 1u << 1,
FirstAndSecond = First | Second,
};
chi::Flags<MyOpts> fl{MyOpts::First | MyOpts::Second};
assert(fl & MyOpts::First);
